# Pyrgos: TowerFall Mods Launcher

Pyrgos is a launcher that let you patch TowerFall with mods powered by FortRise. 

**Features:**
+ Vanilla and Modded launching
+ Installers
+ Client Manager
+ Mod List

**This Launcher is currently work in progress**
